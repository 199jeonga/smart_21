데이터타입 (형타입)

  • 숫자형 : js에서는 실수, 정수 구분없이 숫자라고 표현함
  • 문자형 : 문자는 반드시 '' 내부에 작성하는 형태이며 숫자를 작성하더라도 따옴표 내부에 있으면 문자이다.
  • 논리형 : 참(true) or 거짓(false)를 의미하는 결과물을 도출 - true, false는 문자가 아니다. 따옴표 금지
  • undefined : 값이 지정되지 않은 상태, 변수나 그무엇은(이게뭔데) 존재/유지
  • null : 값이 없기나 값 전달 시 없는 값
