양방향 전송방식: 입력 - 보내기 - 받아서 처리 - 결과 피드백

검색, 로그인, 회원가입, 구매, 게시판

  1. 검색하기 위해 필요한 단어, 문장 작성 (글을 작성하는 영역)
  2. 전송버튼 클릭 (데이터를 보내는 능력을 가진 버튼 필요)
  3. 테이터 전송 (보낼 곳 확인, 노출/비공개/이건 지금 못함:삭제/수정 등의 옵션)
  4. 전송 받은 곳에서 수행 (백엔드)
  5. 수행에 대한 결과 피드백
  6. 피드백받은 결과 도출
